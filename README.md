# CoursesStack

iOS app for course publishing and learning workflows.

## Features

- Course creation and management
- Lessons, assignments, and access control
- Course purchase/consumption flows
- Reviews and ratings
- RU/EN localization

## Stack

- SwiftUI
- Firebase

## Run locally

1. Clone the repository:
```bash
git clone https://github.com/NataliaAtiukova/CoursesStack.git
cd CoursesStack
```
2. Open project in Xcode.
3. Add your Firebase `GoogleService-Info.plist`.
4. Build and run.
